## 通过nvm安装（推荐）

### 1、安装nvm

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
```

### 2、使用nvm安装指定版本 Node.js
```bash
nvm install 22
```

### 3、查看Node版本号
```bash
node -v # should print `v22.2.0`
```
